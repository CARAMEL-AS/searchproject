CONFIGURE YOUR ENV FILE FOR GOOGLE MAPS
1. create .env file in project root folder
2. create variable REACT_APP_GOOGLE_API='' inside .env file
3. add your personal API key from Google Console into ''
4. run your project using npm start or yarn start
